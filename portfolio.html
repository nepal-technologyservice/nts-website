Migrease

A full-stack web platform built and deployed to production on AWS infrastructure. Migrease was built from the ground up with proper cloud architecture — serverless Lambda functions, S3, CloudFront, and a complete CI/CD pipeline with GitHub Actions. The frontend runs on Next.js, the backend on Node.js with TypeScript. Everything from the Terraform infrastructure to the API design follows international deployment standards.
